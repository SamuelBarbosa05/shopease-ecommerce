# 🛍️ ShopEase E-commerce

Aplicação de e-commerce desenvolvida com **HTML, CSS e JavaScript puro**, focada em simular uma experiência real de loja online com carrinho de compras, filtros e persistência de dados.

## ✨ Funcionalidades

- 🔍 Busca de produtos em tempo real  
- 🏷️ Filtro por categoria  
- 🛒 Adicionar produtos ao carrinho  
- ➕ Aumentar / ➖ diminuir quantidade  
- ❌ Remover itens do carrinho  
- 💰 Cálculo automático do total  
- 💾 Persistência com **localStorage**  
- 📱 Layout responsivo  

---

## 🧠 Tecnologias utilizadas

- HTML5  
- CSS3  
- JavaScript (ES6+)  
- DOM Manipulation  
- localStorage
